apple/container

⭐ 35,170 · 今日 +3,504 · Swift

Apple 官方推出的 Linux 容器运行工具，基于轻量级虚拟机在 Mac 上直接运行 OCI 容器，针对 Apple Silicon 深度优化。需 macOS 26 及以上版本。

面向 AI 编码助手的工程级技能集合，覆盖需求访谈、规格编写、任务拆解、增量实现、TDD、代码审查、安全加固、性能优化、CI/CD 等 24 个完整开发流程技能。

obra/superpowers

⭐ trending · MIT · Multi-agent framework

一套完整的 AI 编码助手开发方法论，通过可组合的技能模块实现自动化的头脑→规格→计划→子代理执行→代码审查→合并工作流。支持 Claude Code、Codex、Gemini CLI、Cursor 等。

值得关注：核心创新是"子代理驱动开发"（Subagent-Driven Development）——每个任务由独立子代理执行并经过两轮审查（规格符合性 + 代码质量），AI 可连续自主工作数小时不偏离计划。与 agent-skills 思路相近但更侧重多代理编排。

refactoringhq/tolaria

⭐ 15,788 · 今日 +369 · TypeScript

跨平台桌面应用（macOS / Windows / Linux），用于管理 Markdown 知识库。文件优先（纯 .md）、Git 优先、离线优先，支持为 AI 助手组织上下文文档。
