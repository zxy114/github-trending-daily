⭐ 38,828 📈 +4,005 今日 🐍 Python
在工具输出、日志、文件和 RAG 片段送入 LLM 之前进行智能压缩，减少 60-95% 的 token 消耗，同时保持回答质量不变。支持库、代理和 MCP 服务器三种模式。
💡 入选理由：单日 4000+ 星的增长说明社区对 LLM 成本优化的迫切需求。对于大量调用 API 的项目，token 节省可以直接转化为真金白银。

iroh — 基于密钥的模块化网络栈

⭐ 10,255 📈 +302 今日 🦀 Rust
IP 地址会变化，用密钥拨号代替。基于 Rust 实现的模块化网络栈，通过密钥标识节点而非依赖 IP 地址，解决网络地址不稳定的问题。
💡 入选理由：「用密钥代替 IP」是 P2P 网络的重要思路，对于构建去中心化应用和移动网络环境下的可靠通信有参考价值。Rust 实现保证性能和安全。
