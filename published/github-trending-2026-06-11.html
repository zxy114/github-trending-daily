🍎 apple/container — Apple 官方 Linux 容器工具

⭐ 29,800+ stars 📈 +1,611 stars/day 🔖 Swift
Apple 官方开源的 Linux 容器管理工具，专为 Apple Silicon Mac 设计。利用轻量级虚拟机在 macOS 上运行 Linux 容器，无需 Docker Desktop。Swift 编写，充分利用 Apple Silicon 的虚拟化框架。这是 Apple 首次将此类基础设施级工具开源，标志着 Mac 开发者体验的重要变化。

📡 RuView — 用 WiFi 信号做空间感知与生命体征监测

📈 695 stars/day 🔖 Python / Rust
通过 ESP32 捕获 WiFi 信道状态信息（CSI），将普通 WiFi 信号转化为空间感知系统。可实现穿墙人体检测、姿态估计（17关键点）、呼吸频率与心率实时监测、跌倒检测等。模型仅 8KB（4-bit 量化），在树莓派上微秒级推理。已集成 Home Assistant、Apple Home、Google Home、Alexa、Matter 等主流智能家居平台。适合养老监护、零售客流、办公空间管理等场景，无需摄像头，避免隐私问题。

🏥 OpenMed — 开源医疗 AI 平台

⭐ 2,320+ stars 📈 +527 stars/day 🔖 Python
面向医疗行业的开源 AI 框架，覆盖影像分析、基因组学、电子病历处理等多模态场景。提供开箱即用的预训练模型和可配置流水线，支持临床研究与辅助诊断。增长迅速，社区活跃。

🔍 Roboflow Supervision — 计算机视觉工具库

📈 热门 CV 库 🔖 Python
Roboflow 出品的可复用计算机视觉工具集。提供检测框/分割掩码标注、目标追踪、区域计数、热力图、置信度过滤等丰富的后处理与可视化能力，兼容 YOLO、SAM、CLIP 等主流模型。是构建 CV 应用的"瑞士军刀"，降低从模型输出到业务逻辑的开发成本。
